# Password Manager using Python

## Overview

This project is a simple Password Manager developed using Python as part of an Industrial Internship. It allows users to securely store, view, update, search, and delete password records through a menu-driven interface.

## Features

* Add Password
* View Passwords
* Search Password
* Update Password
* Delete Password
* File-based Storage

## Technologies Used

* Python
* NumPy
* Pandas
* File Handling

## How to Run

1. Install Python.
2. Install required libraries:

   ```
   pip install numpy pandas
   ```
3. Run:

   ```
   python main.py
   ```
